## 📦 Installation Guide

### System Requirements
- **Windows**: 10/11 (x64), 4GB RAM, 500MB disk space
- **macOS**: 10.15+ (Intel/Apple Silicon), 4GB RAM, 500MB disk space
- **Linux**: Ubuntu 20.04+/CentOS 8+ (x64), 4GB RAM, 500MB disk space

### Installation Methods

#### Method 1: One-Click Installer (Recommended)
```bash
# Windows (PowerShell as Administrator)
Invoke-WebRequest -Uri "https://github.com/apollo-shield/releases/apollo-installer.bat" -OutFile "apollo-installer.bat"
.\apollo-installer.bat

# macOS/Linux
curl -fsSL https://github.com/apollo-shield/releases/install.sh | sudo bash
```

#### Method 2: Direct Download
1. Download from [GitHub Releases](https://github.com/apollo-shield/desktop-app/releases)
2. **Windows**: Run `Apollo-Setup-1.0.0-x64.exe`
3. **macOS**: Open `Apollo-1.0.0-arm64.dmg`
4. **Linux**: Run `Apollo-1.0.0-x64.AppImage`

#### Method 3: Build from Source
```bash
git clone https://github.com/apollo-shield/desktop-app.git
cd desktop-app
npm install
npm run build
```

### Post-Installation
1. **Launch Apollo** from desktop shortcut or Start Menu
2. **Protection activates automatically** - no configuration needed
3. **System tray icon** indicates active protection
4. **Dashboard accessible** via tray icon or desktop shortcut
