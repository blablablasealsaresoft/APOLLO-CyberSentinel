H.1 NIST SP 800-86 Order of Volatility

Evidence Collection Sequence

  1. CPU State: Registers & cache (most volatile - immediate capture)
  2. Memory Dump: RAM contents & running processes (0-30 seconds)
  3. Network State: Connections & routing tables (0-30 seconds)
  4. Process State: Running processes & loaded modules (0-5 minutes)
  5. File System: Metadata & temporary files (persistent)
  6. Registry: Configuration data (persistent)
  7. System Logs: Audit trails (least volatile)
